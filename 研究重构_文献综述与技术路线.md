# 面向教育资源检索的可信多模态知识推理研究：文献综述与研究内容重构

版本：v0.1  
日期：2026-06-16  
定位：用于重新确定硕士论文的研究问题、理论依据、技术路线和实验设计。

## 0. 总体研究主旨

原始主旨：

> 在教育资源检索中，需要解决跨模态数据对齐、模态缺失情况下的推理稳定性以及推理过程的可解释性问题。

建议重构为：

> 面向教育资源检索的可信多模态知识推理研究：围绕“资源是否对齐”“缺失是否可信”“解释是否忠实”三个问题，构建跨模态资源对齐、非随机缺失建模与图约束解释生成方法。

三个研究内容之间的关系应为：

```text
研究内容一：教育知识结构约束的跨模态资源对齐
    输出：resource-concept alignment score

研究内容二：面向非随机模态缺失的选择性因果补全
    输出：modality status + reliability score

研究内容三：融合路径、对齐与模态可靠性的图约束解释生成
    输入：alignment score + modality reliability + graph path
    输出：faithful educational explanation
```

注意：三个研究内容不应被强行写成一个“大一统模型”。更稳妥的论文结构是三个递进科学问题，每个问题都有独立文献支撑、方法、实验与失败风险。

## 2. 研究内容一：教育知识结构约束的跨模态资源对齐

### 2.1 现有研究缺口

通用视觉-语言模型已经能做 image-text alignment，但它解决的是“图文语义是否相似”。教育资源检索真正需要的是“资源是否对应同一知识点、同一认知粒度和同一教学目标”。

已有工作的问题：

1. CLIP/BLIP/SigLIP 主要学习开放域图文相似性，不显式建模教育知识点。
2. GFM 关注跨图泛化和结构适配，但直接把图卷积叠加到跨模态对齐任务上可能造成结构噪声。
3. MMKG 研究多关注实体对齐、链接预测或通用 VQA，较少针对教育资源检索中的 concept-level hard negative。

### 2.2 科学问题

**SQ1：教育知识结构能否作为跨模态对齐的约束，缓解通用视觉-语言模型在教育资源检索中的概念粒度错配？**

进一步拆分：

- SQ1.1：通用 CLIP 在 ScienceQA 等教育数据中是否存在概念粒度错配？
- SQ1.2：同 topic / 邻近 topic 的 hard negative 是否比随机负样本更能提升教育资源检索？
- SQ1.3：知识结构应作为 message passing、loss constraint 还是 negative sampler 才最有效？

### 2.3 技术路线

建议方法名：

> Concept-Structure-Aware Cross-Modal Alignment, CS-CMA

整体流程：

```text
教育资源 r = {text, image, lecture, topic, skill}
        ↓
通用编码器：
    text/image → CLIP 或 SigLIP embedding
        ↓
资源-概念映射：
    resource → concept/topic/skill
        ↓
教育 hard negative 构造：
    positive: same resource text-image pair
    easy negative: random different topic
    hard negative: same subject / related topic / adjacent concept
        ↓
结构约束对比学习：
    L_itc: image-text contrastive loss
    L_rc: resource-concept alignment loss
    L_hn: hard-negative margin loss
    L_hierarchy: concept hierarchy regularization
        ↓
输出：
    教育语义空间下的 resource embedding
```

### 2.4 推荐模型结构

不建议从头训练 GFM。建议用轻量 adapter：

```text
Frozen CLIP/SigLIP encoder
        ↓
Modality-specific adapter
        ↓
Shared educational semantic space
        ↓
Contrastive + hard negative + concept regularization
```

图结构的使用方式：

- 推荐：作为 hard-negative sampler 和 regularization。
- 谨慎：直接 GraphSAGE message passing。
- 原因：图结构质量不高时，message passing 会污染跨模态空间。

### 2.5 Baseline

必须对比：

1. CLIP zero-shot retrieval
2. CLIP + linear adapter
3. CLIP + ordinary contrastive fine-tuning
4. CLIP + random hard negatives
5. CLIP + concept-aware hard negatives
6. CLIP + concept-aware hard negatives + hierarchy loss
7. Graph message passing variant

### 2.6 指标

核心指标：

- Text-to-image Recall@K
- Image-to-text Recall@K
- MRR
- Concept-level retrieval accuracy
- Hard-negative error rate

建议新增诊断指标：

- Same-topic confusion rate
- Adjacent-topic confusion rate
- Modality gap before/after adaptation

### 2.7 可失败点

如果 concept-aware hard negative 无提升，说明 ScienceQA 的 topic 粒度不适合作为知识结构。备用方案：

- 引入 skill 字段；
- 引入 OpenStax 教材章节结构；
- 人工构建小规模 prerequisite graph。

## 3. 研究内容二：面向非随机模态缺失的选择性因果补全

### 3.1 现有研究缺口

已有 missing modality 方法大多假设：

> 模态缺失是一种需要鲁棒处理或补全的缺陷。

但教育资源中的缺失可能具有结构性：

- 语法、拼写、阅读理解类资源天然不一定需要图像；
- 几何、物理实验、生物结构类资源缺图可能是严重信息损失；
- 历史、地理类资源是否需要图像取决于具体知识点。

因此，教育场景下更关键的问题不是“如何补全所有缺失模态”，而是：

> 哪些缺失应该补，哪些缺失不应该补？

### 3.2 科学问题

**SQ2：教育多模态资源中的模态缺失是否具有知识点依赖的 MNAR / structured missingness 特征？如果有，如何进行选择性补全以避免教育幻觉？**

进一步拆分：

- SQ2.1：图像缺失率是否显著依赖 topic / subject / skill？
- SQ2.2：能否区分 accidental missing、structural absence 和 ambiguous missing？
- SQ2.3：选择性补全是否比无差别补全更能提升缺失条件下的检索稳定性？

### 3.3 技术路线

建议方法名：

> MNAR-Aware Selective Modality Completion, MAS-MC

整体流程：

```text
教育资源：text, topic, subject, skill, image_missing
        ↓
Missingness Mechanism Estimator
    输入：topic/subject/skill/text embedding
    输出：P(M_image=missing | C, X_text)
        ↓
缺失类型判别：
    accidental missing
    structural absence
    ambiguous missing
        ↓
Selective Completion
    accidental → 补全或检索替代图像
    structural → 不补全，标记为 structure absence
    ambiguous → 低置信度处理
        ↓
下游资源检索：
    使用 modality reliability 修正 ranking
```

### 3.4 因果建模

可以建立 SCM：

```text
C = concept/topic/skill
T = text representation
V = visual representation
M = modality missingness indicator
Y = 检索/推荐目标

C → T
C → V
C → M
T → V
V → Y
T → Y
M → observed representation
```

关键不是简单生成 `V_hat`，而是建模：

```text
P(M | C, T)
P(type_missing | C, T, graph_context)
P(V | T, do(C), type=accidental)
```

### 3.5 Baseline

必须对比：

1. Zero imputation
2. Mean imputation
3. Vanilla VAE imputation
4. Modality dropout / missing-modality robust fusion
5. SimMLM-like dynamic modality experts
6. Causal imputation without selective gate
7. Ours: selective causal completion

### 3.6 指标

不要以 CosSim 作为唯一指标。建议：

#### 缺失机制指标

- 缺失预测 AUC
- Topic 条件缺失熵
- Topic 与缺失之间的卡方检验 / 互信息

#### 缺失类型指标

- Structural absence 检测准确率
- Accidental missing detection accuracy

如果没有人工标签，可先用 LLM 辅助标注 300-500 条，再人工抽检。

#### 下游指标

- Retrieval Recall@K under missing modality
- Ranking robustness under modality dropout
- Hallucinated visual completion rate
- High-MNAR group vs low-MNAR group performance gap

### 3.7 可失败点

如果没有缺失类型 ground truth，选择性补全难以严格验证。备用方案：

- 用 topic-level missing rate 构造弱标签；
- 让 LLM 标注“该题是否需要图像辅助理解”；
- 人工抽样 200 条验证。

## 4. 研究内容三：融合路径、对齐与模态可靠性的图约束解释生成

### 4.1 现有研究缺口

KG + LLM 已经有人做。单纯把路径塞进 prompt 不再有创新性。

现有工作的核心趋势是：

- 通过 KG 路径增强 LLM 推理；
- 约束 LLM 输出，降低幻觉；
- 用 KG triples 检查生成内容是否忠实。

但教育资源推荐解释还有一个额外需求：

> 解释不仅要忠实于知识路径，还要说明资源本身是否跨模态一致、模态是否完整、补全是否可靠。

### 4.2 科学问题

**SQ3：如何让教育资源推荐解释同时满足图谱路径忠实性、跨模态证据一致性和教学可操作性？**

进一步拆分：

- SQ3.1：KG path 能否降低 LLM 推荐解释中的关系幻觉？
- SQ3.2：alignment evidence 能否减少“推荐资源与知识点不一致”的解释错误？
- SQ3.3：modality reliability 是否能提高用户对解释的信任和可操作性？

### 4.3 技术路线

建议方法名：

> Path-Alignment-Reliability Grounded Explanation, PAR-GE

整体流程：

```text
学生查询 / 目标知识点 / 当前掌握状态
        ↓
概念识别：
    查询 → 目标概念
        ↓
路径检索：
    当前概念 → 先修/补救路径 → 目标概念
        ↓
资源召回：
    每个 concept 检索文本、图像、视频资源
        ↓
证据包构建：
    路径证据
    RC1 对齐分数
    RC2 模态状态
    可靠性评分
        ↓
LLM 约束解释
        ↓
验证器：
    概念是否存在？
    关系是否存在？
    资源是否存在？
    陈述是否有证据支持？
```

### 4.4 解释证据包格式

每个候选推荐应包含：

```json
{
  "path": [
    {"source": "force", "relation": "prerequisite_of", "target": "acceleration"},
    {"source": "acceleration", "relation": "prerequisite_of", "target": "newton_second_law"}
  ],
  "resources": [
    {
      "resource_id": "r_001",
      "concept": "acceleration",
      "alignment_score": 0.82,
      "modality_status": "original_multimodal",
      "reliability": 0.91
    }
  ],
  "student_state": {
    "mastered": ["force"],
    "weak": ["acceleration"]
  }
}
```

### 4.5 Baseline

必须对比：

1. 仅 LLM 解释
2. 文本 RAG 解释
3. KG 路径提示解释
4. 路径 + 对齐证据
5. 路径 + 模态可靠性
6. Full PAR-GE

### 4.6 指标

自动指标：

- Path Faithfulness：解释中的关系是否存在于图谱。
- Entity Grounding：解释中的概念/资源是否存在于证据包。
- Unsupported Claim Rate：无法由证据包支持的陈述比例。
- Hallucination Rate：图谱外实体或资源比例。
- Evidence Coverage：解释是否覆盖关键路径节点和资源证据。

人工指标：

- 教学合理性
- 个性化程度
- 可操作性
- 可信度

### 4.7 可失败点

如果图谱路径质量差，图约束会导致解释僵硬甚至错误。备用方案：

- 先做小规模高质量 concept graph；
- 限定关系类型：prerequisite、related、remedial；
- 用 verifier 过滤低质量路径。

```text
RC1：跨模态对齐
    concept-aware hard negative contrastive learning
    输出 alignment score
        ↓

RC2：缺失模态建模
    missingness mechanism estimation
    selective causal completion
    输出 modality status + reliability
        ↓

RC3：可解释推理
    path retrieval
    证据包
    constrained LLM explanation
    verifier
        ↓

评估层：
    retrieval performance
    missingness robustness
    explanation faithfulness
```

## 6. 推荐论文贡献写法

最终论文贡献不要写得过大。建议写成：

1. **提出教育知识结构约束的跨模态资源对齐方法**，通过 concept-aware hard negatives 和 hierarchy-aware regularization 缓解通用视觉-语言模型在教育资源检索中的概念粒度错配。

### 11.1 研究内容一相关：跨模态对齐、MMKG 与 GFM/GraphLLM

#### 11.1.1 通用视觉-语言对齐与 modality gap

| 文献                     | 方法核心                                        | 可作为         | 与本研究的关系                          |
| ---------------------- | ------------------------------------------- | ----------- | -------------------------------- |
| CLIP, 2021             | 大规模 image-text contrastive pretraining      | 必须 baseline | 教育资源检索的通用跨模态基线                   |
| Mind the Gap, 2022     | 从理论和实证角度说明 CLIP 表征中存在 modality gap          | 理论支撑        | 支撑“通用多模态模型在领域迁移中仍需再对齐”           |
| Multimodal-CoT, 2023   | 在 ScienceQA 上使用 text+vision rationale 两阶段推理 | 教育多模态推理参考   | 说明 ScienceQA 不只是检索数据，也可用于解释/推理评估 |
| Graph-of-Thought, 2023 | 将推理链扩展为图结构，在 ScienceQA 上验证                  | 推理结构参考      | 可为 RC3 的图约束解释提供背景                |

写作要点：

> 通用视觉-语言模型通过大规模对比学习建立图文语义空间，但已有研究表明其表示空间存在 modality gap。对于教育资源检索而言，图文语义相似并不等价于教学目标一致，因此需要引入教育知识结构来约束对齐过程。

#### 11.1.2 多模态知识图谱与 MMKG completion

| 文献                                                     | 方法核心                                                                           | 可作为                  | 局限                             |
| ------------------------------------------------------ | ------------------------------------------------------------------------------ | -------------------- | ------------------------------ |
| MKGformer, 2022                                        | hybrid transformer + multi-level fusion，统一处理多模态 link prediction、RE、NER         | MMKG baseline/参考     | 偏通用 MMKG completion，不面向教育检索    |
| NativE, 2024                                           | 面向真实 MMKG 中多模态类型不均衡和模态缺失，提出 relation-guided adaptive fusion                    | 强相关 baseline 思路      | 目标是 KG completion，不区分教育中的结构性缺失 |
| MMPKUBase, 2024                                        | 构建中文多模态 KG，并用 prototypical contrastive learning 清洗图像                           | 数据构建参考               | 非教育领域                          |
| MR-MKG, 2024                                           | 用 MMKG 增强 LLM 多模态推理，包含 relation graph attention 和 cross-modal alignment module | RC1/RC3 重要参照         | 不是教育场景，且不处理学习路径解释              |
| Transformer-based MMKGC with Link-Aware Contexts, 2025 | 用 VLM 将视觉邻域上下文转为文本，再用 seq2seq 做 MMKGC                                          | 方法启发                 | 更偏 KG link prediction          |

| MMCOMET, 2026                                          | 大规模 multimodal commonsense KG，通过图像检索扩展 ATOMIC2020                              | MMKG 构建参考            | 常识图谱，不是教育图谱                    |

写作要点：

> MMKG completion 已经研究了如何融合图结构和多模态信息，但多数工作以实体补全或链接预测为目标。教育资源检索的核心不是补全 KG 三元组，而是判断某个文本/图像/视频资源是否对应同一个教学概念和认知目标。因此，本研究应将 MMKG 方法中的 adaptive fusion、negative sampling、alignment module 转化为教育资源对齐任务，而不是直接复现 MMKGC。

#### 11.1.3 GFM、GraphLLM 与图泛化

| 文献              | 方法核心                                                         | 可作为         | 对本研究的启发                       |
| --------------- | ------------------------------------------------------------ | ----------- | ----------------------------- |
| OpenGraph, 2024 | LLM 增强图数据、统一 graph tokenizer、scalable graph transformer      | GFM 代表方法    | GFM 的目标是跨图泛化，不是简单 GNN         |
| AnyGraph, 2024  | Graph MoE 处理 structure heterogeneity 和 feature heterogeneity | GFM 代表方法    | 支撑“图结构和特征异构是核心问题”             |
| GraphGPT, 2023  | graph instruction tuning，让 LLM 理解图结构                         | GraphLLM 参考 | 可用于 RC3 的图路径语言化               |
| GraphLLM, 2023  | 用 graph learning model 增强 LLM graph reasoning                | GraphLLM 参考 | 支撑“Graph2Text 直接描述图可能有瓶颈”     |
| LLaGA, 2024     | structure-aware sequence + graph-text projector              | GraphLLM 参考 | 可以借鉴“图结构转 token embedding”思想  |
| GLBench, 2024   | GraphLLM 统一评测，指出 LLM-as-enhancer 更稳，LLM-as-predictor 可控性差    | 评估警示        | RC3 不应让 LLM 自由预测，应让 LLM 做受控解释 |

> GFM/GraphLLM 最新工作关注图结构异构、特征异构、跨图迁移和图到语言的接口问题。对本研究而言，图不应被简单作为 message passing 模块叠加到跨模态 embedding 上，而应作为教育 hard negative 构造、概念层级正则和解释约束来源。

### 11.2 研究内容二相关：缺失模态、MNAR 与选择性补全

#### 11.2.1 传统 missing modality 方法

| 文献                                                      | 方法核心                                                            | 可作为 baseline                     | 局限              |
| ------------------------------------------------------- | --------------------------------------------------------------- | -------------------------------- | --------------- |
| Modality Dropout / Unified Representation Network, 2019 | 训练时随机丢弃模态，学习可变模态输入                                              | 基础 baseline                      | 默认缺失是随机或可模拟     |
| LRMM, 2018                                              | 推荐系统中使用 modality dropout 和 multimodal autoencoder 补全缺失模态        | 推荐场景 baseline                    | 不处理教育知识点依赖缺失    |
| M3AE, 2023                                              | multimodal masked autoencoder，自监督处理缺失模态                         | 强 baseline                       | 医学图像分割，非教育资源    |
| ModDrop++, 2022                                         | dynamic filter + intra-subject co-training                      | missing modality robust baseline | 医学多模态，缺失机制与教育不同 |

写作要点：

> 现有 missing modality 方法主要解决“缺失时如何保持预测性能”，常见策略包括 modality dropout、masked autoencoding、virtual modality generation 和 dynamic modality experts。但这些方法通常默认缺失模态应被补全或鲁棒融合，没有回答教育资源中“某些知识点本身不需要视觉模态”的结构性缺失问题。

#### 11.2.2 结构化缺失与因果 missingness

| 文献                                                 | 方法核心                                    | 可作为理论支撑 | 对本研究的启发              |
| -------------------------------------------------- | --------------------------------------- | ------- | -------------------- |
| Learning from Structured Missingness, 2023         | 将 structured missingness 作为异构数据学习中的基础问题 | 理论支撑    | 教育资源缺图可能是结构性信号       |
| Deep ML with Missing Modality Survey, 2024         | 系统总结 deep missing modality 方法、应用和挑战     | 综述支撑    | 帮助定位 RC2 在 MLMM 中的位置 |
| Causal Debiasing Medical Multimodal Learning, 2025 | 处理 missingness bias 和 latent confounder | 因果支撑    | 支撑“非随机缺失会带来偏差”       |
| MMNAR Clinical Representation, 2025                | missingness pattern informative，不应仅当作噪声 | 因果支撑    | 支撑“缺失机制本身应建模”        |

写作要点：

> 医疗多模态领域已经开始将 non-random modality missingness 视为偏差来源和信息来源，但教育多模态资源中尚缺少类似建模。教育场景的独特性在于，知识点类型本身决定了视觉模态的必要性，因此需要从“盲目补全”转向“选择性补全”。

### 11.3 研究内容三相关：KG-grounded LLM、GraphRAG 与解释忠实性

#### 11.3.1 教育推荐解释中的 KG + LLM

| 文献                                                                                               | 方法核心                                                           | 与本研究关系   | 局限                                 |
| ------------------------------------------------------------------------------------------------ | -------------------------------------------------------------- | -------- | ---------------------------------- |
| Knowledge Graphs as Context Sources for LLM-Based Explanations of Learning Recommendations, 2024 | 将 KG 作为 LLM prompt 的 factual context，减少教育推荐解释中的不精确内容           | 最直接相关工作  | 偏 prompt + template，未引入跨模态对齐和模态可靠性 |
| LLM-Based Chatbot with KG Contextualization for Learning Recommendations, 2024                   | 用 KG contextualization 支持学习路径推荐的 conversational explainability | 教育解释直接相关 | 概念验证为主，缺少严格路径忠实性和多模态证据评估           |

写作要点：

> 已有教育工作已经将 KG 作为 LLM 解释的上下文来源，因此本研究不能只做“KG + prompt”。应进一步引入跨模态对齐证据和模态可靠性证据，并通过 verifier 检查解释是否忠实。

#### 11.3.2 KG-grounded faithful reasoning

| 文献                                | 方法核心                                                    | 可借鉴点                            | 局限          |
| --------------------------------- | ------------------------------------------------------- | ------------------------------- | ----------- |
| Graph-constrained Reasoning, 2024 | 用 KG-Trie 将 KG reasoning paths 编码进 decoding，保证生成路径图谱可验证 | 强支撑：解释必须受图约束                    | 实现复杂；偏 KGQA |
| Paths-over-Graph, 2024            | 动态探索 KG reasoning paths，并用 LLM 进行路径推理                   | 路径检索与 pruning                   | 通用 KGQA，非教育 |
| Graph-Augmented Reasoning, 2025   | step-by-step KG retrieval + reasoning                   | 分步学习路径启发                        | 数学推理，不是推荐解释 |
| GraphEval, 2024                   | 将文本响应转为 KG triples，用 KG/NLI 检测 hallucination            | 可作为解释评估方法                       | 不直接生成解释     |
| MultiHal, 2025                    | KG-grounded multilingual hallucination evaluation       | 可作为 hallucination evaluation 参照 | 通用事实问答      |

写作要点：

> KG-grounded LLM 的前沿已从“把知识塞进 prompt”转向“约束推理路径”和“评估事实忠实性”。因此，本研究的 RC3 应强调路径忠实性、证据对齐和无依据陈述检测。

#### 11.3.3 RAG vs GraphRAG 的负面证据

| 文献                                                | 发现                                                                  | 对本研究的警示                           |
| ------------------------------------------------- | ------------------------------------------------------------------- | --------------------------------- |
| RAG vs. GraphRAG, 2025                            | GraphRAG 并不总是优于 RAG，不同任务收益不同                                        | 必须证明图结构对教育解释有真实增益                 |
| Comparing RAG and GraphRAG on Math Textbook, 2025 | 在数学教材 page-level QA 中，embedding RAG 优于 GraphRAG；GraphRAG 可能检索过多无关内容 | 教育场景下图结构可能引入噪声，需要精细路径过滤和 verifier |

写作要点：

> 最新教育 GraphRAG 研究已经显示，图增强检索并不天然优于向量检索。你的论文必须避免“图一定更好”的假设，而应证明：在解释生成场景中，图路径不是为了提升召回，而是为了提升解释忠实性和可验证性。

## 12. Related Work 章节建议结构

建议 Related Work 不要按技术名罗列，而按科学问题组织：

### 12.1 Multimodal Alignment for Educational Resources

写作逻辑：

1. CLIP/BLIP/SigLIP 等通用视觉-语言模型建立了跨模态语义空间。
2. 但 modality gap 研究表明，图像和文本仍存在空间分离和领域迁移问题。
3. MMKG 和 GFM 工作尝试引入图结构，但多数面向通用实体补全、图泛化或 KGQA。
4. 教育资源检索需要 concept-level alignment 和 teaching-objective alignment，因此需要教育知识结构约束的 hard negative 和 concept regularization。

### 12.2 Missing Modality and Structured Missingness

写作逻辑：

1. Missing modality 已有大量鲁棒融合、模态补全、masked autoencoder 和 dynamic experts 方法。
2. 这些方法大多默认缺失是随机扰动或所有缺失都需要补全。
3. Structured missingness 和 causal missingness 工作表明，缺失机制本身可能携带信息，并引入偏差。
4. 教育资源中的缺图往往依赖知识点类型，因此需要 MNAR-aware selective completion。

### 12.3 KG-grounded LLM Explanation and Faithful Reasoning

写作逻辑：

1. LLM 可生成自然语言解释，但在教育等高可信场景中存在幻觉风险。
2. KG 作为 LLM context 已用于学习推荐解释，但多停留在 prompt/template 层面。
3. KG-grounded faithful reasoning 最新工作强调图约束解码、路径推理和 hallucination evaluation。
4. 本研究进一步将 path、alignment 和 modality reliability 作为可验证证据包，约束教育推荐解释。

## 14. 最应优先验证的三个命题

### 命题一：教育 hard negative 是否真的有效？

如果 concept-aware hard negative 不优于 random negative，则 RC1 创新点不足。

最小实验：

```text
CLIP
CLIP + random contrastive
CLIP + same-subject hard negative
CLIP + same-topic / adjacent-topic hard negative
```

### 命题二：ScienceQA 中图像缺失是否具备强 MNAR？

如果 topic 与 image missingness 之间没有显著关系，则 RC2 不能成立。

最小实验：

```text
M_image ~ topic + subject + skill + text length
指标：AUC, mutual information, chi-square, group missing rate
```

### 命题三：图约束是否降低解释幻觉？

如果 KG-path prompt 相比 LLM-only 没有降低 hallucination rate，则 RC3 需要调整。

最小实验：

```text
LLM-only
RAG
KG-path
KG-path + alignment + modality reliability

指标：
entity grounding
relation grounding
unsupported claim rate
human usefulness
```

## 16. 论文 Related Work 可写段落草稿

### 16.1 跨模态教育资源对齐

通用视觉-语言模型（如 CLIP、BLIP、SigLIP）通过大规模图文对比学习显著提升了跨模态检索能力。然而，已有研究指出，图像和文本嵌入即便位于共享空间中，也可能存在明显的 modality gap，这会影响下游检索和跨域迁移。在教育资源检索中，该问题进一步表现为概念粒度错配：两个资源可能在视觉或文本语义上相似，却对应不同知识点、认知层级或教学目标。

多模态知识图谱和图基础模型为引入结构知识提供了可能。MMKG 方法已经在实体补全、多模态推理和链接预测中证明了图结构与多模态信息融合的价值；GFM 和 GraphLLM 则进一步关注图结构异构、特征异构和跨图泛化。然而，这些方法大多面向通用图任务或知识图谱补全，并未直接解决教育资源检索中的资源-概念对齐问题。因此，本研究关注如何将教育知识结构转化为 hard negative、层级约束和 resource-concept regularization，而不是简单地将图卷积叠加到跨模态表示上。

### 16.2 非随机缺失与选择性补全

多模态学习中的缺失模态问题已受到广泛关注。已有方法包括 modality dropout、missing modality imagination、masked autoencoding、virtual modality generation 和 dynamic modality experts。这些方法通常假设缺失模态代表信息不完整，因此目标是恢复缺失信息或学习对缺失鲁棒的融合表示。

然而，结构化缺失和因果缺失研究表明，缺失机制本身可能携带重要信息，并可能引入偏差。在教育资源中，图像缺失往往与知识点类型相关：几何、物理实验和生物结构类知识点缺图可能造成严重理解障碍，而语法、词汇或阅读理解类知识点天然不一定需要图像。因此，本研究将缺失模态问题重新定义为 MNAR-aware selective completion：系统首先判断缺失属于 accidental missing、structural absence 还是 ambiguous missing，再决定是否进行补全。

### 16.3 图约束解释生成

LLM 能生成流畅的自然语言解释，但在教育推荐等高可信场景中，解释必须忠实于真实知识结构和资源证据。已有研究已经开始将 KG 作为 LLM 解释学习推荐的上下文来源，以降低错误和不精确信息。然而，单纯将 KG triples 或路径放入 prompt 并不能保证解释忠实性。

KG-grounded faithful reasoning 的最新工作进一步提出图约束解码、路径检索和 hallucination evaluation，强调生成内容应能回溯到知识图谱中的实体、关系和路径。同时，GraphRAG 的系统评估显示，图结构并不总能提升检索性能，低质量图甚至可能引入噪声。因此，本研究将图谱路径用于解释约束和忠实性验证，而不是假设其必然提升检索召回；并进一步融合跨模态对齐得分和模态可靠性，构建可验证的教育推荐解释。
